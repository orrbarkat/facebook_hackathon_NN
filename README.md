# facebook_hackathon_NN
hashtag classifier

1. clone the repo

2. ```$ virtualev -p python3.6 <path/to/repo>```

3. ```$ cd <path/to/repo> && source bin/activate```

4. ```$ pip install -r requirement.txt```

5. if it's the first time running the project comment out `model = load_model('my_model_2.h5')`
enjoy!
